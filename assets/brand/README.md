# Placarium Brand Assets

Pasta oficial de assets de marca do Placarium.

## Estrutura

```text
assets/brand/
  app-icon/        icones quadrados para app/PWA/avatar
  icon-small/      icones compactos para favicon e UI pequena
  logo-completa/   simbolo + wordmark
  preview/         board visual para revisao
  simbolo/         apenas o monograma/grafico
  wordmark/        apenas o nome placarium
```

## Uso rapido

- Header dark: `logo-completa/placarium-logo-completa-dark.svg`
- Header light: `logo-completa/placarium-logo-completa-light.svg`
- Favicon dark: `icon-small/placarium-icon-small-dark.svg`
- PWA/app dark: `app-icon/placarium-app-icon-dark.svg`
- Marca isolada: `simbolo/placarium-simbolo-dark.svg`

Os SVGs sao os mestres editaveis. Para exportar PNG, use estes arquivos como
fonte e mantenha os mesmos nomes base.
